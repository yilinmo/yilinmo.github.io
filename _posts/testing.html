Code blocks

Code blocks can be produced in the usual way with #+BEGINSRC blocks in org-mode. Highlighting is automatic when a pygments lexer is available. 

def foo
  puts 'foo'
end

Same construct with the -n switch, which should turn on line numbers: 

1: def foo
2:   puts 'foo'
3: end

and the same construct with the +n switch, which should continue the linenumbering from the previous snippet 

4: def foo
5:   puts 'foo'
6: end

Other code related embedding is for githubs gist facility

Rules

In orgmode a line on its own with nothing else but at least 5 dashes is considered a rule.

So, when we use 4 we should just get 4 dashes: -—

LaTeX

Orgmode has extensive support for LaTeX fragments, but this doesn't transfer well with the exporter in use.

\begin{equation} x=\sqrt{b} \end{equation}

If \(a^2=b\) and \( b=2 \), then the solution must be either $$ a=+\sqrt{2} $$ or \[ a=-\sqrt{2} \].
